# 웹사이트 PRD (Product Requirements Document): 이상준 개인 포트폴리오

## 1. 개요 (Overview)
- **제품명**: 이상준(Lee Sang-jun) 개인 포트폴리오 웹사이트
- **목적**: '경영학적 비즈니스 이해도'와 '개발/데이터 분석 역량'을 동시에 갖춘 융합형 인재임을 어필하여, 타겟 IT 기업이나 스타트업의 인사 담당자(Recruiter) 및 실무진에게 강렬한 인상을 남기고 채용 제안을 이끌어내기 위함.
- **타겟 오디언스**: IT 프론트엔드/백엔드 개발 팀, 데이터 분석 팀, 채용 담당자 등.
- **핵심 메시지**: *"비즈니스의 흐름(경영 정보)을 이해하고, 데이터(머신러닝/마이닝)로 분석하며, 웹 서비스로 실체화(프론트/백엔드)할 수 있는 예비 전문가"*

## 2. 디자인 요구사항 (Design & UX/UI)
채용 담당자가 사이트에 접속하자마자 "프리미엄(Premium)"과 "현대적(Modern)"이라는 느낌을 받을 수 있도록 디자인합니다.
- **컨셉**: 모던하고 세련된 미니멀리즘 (Sleek, Professional, High-End).
- **테마 & 색상**: 
  - 신뢰감을 주는 다크 모드(Dark Mode) 또는 우아한 무채색 기반의 모던 컬러 팔레트.
  - 단조로운 원색(빨강, 파랑 등)을 피하고 HSL 기반의 테일러된 색상과 부드러운 그라데이션 적용.
- **타이포그래피**: `Pretendard`, `Inter` 등 세련되고 가독성 높은 최신 폰트 활용.
- **인터랙션(Interaction)**: 
  - 유저의 스크롤 위치에 반응하여 요소가 자연스럽게 나타나는 스크롤 애니메이션.
  - 마우스 호버(Hover) 시 반응하는 카드 UI 및 마이크로 인터랙션을 통한 동적이고 살아있는 사용자 경험 제공.
  - 글래스모피즘(Glassmorphism) 효과를 일부 도입해 심도와 고급스러움 연출.

## 4. 기술 스택 및 구현 방향 (Technical Specifications)
- **Core Framework**: React 기반의 `Next.js` 또는 `Vite`를 활용하여 SPA 방식의 빠르고 매끄러운 동작 구현.
- **Styling**: Vanilla CSS, CSS Modules 이용 (또는 Tailwind CSS 활용). 고급스러운 UI 컴포넌트, 애니메이션(Framer Motion 등) 적용.
- **Deployment**: Vercel이나 Netlify를 활용한 빠르고 안정적인 호스팅 반영.
- **SEO & 성능**: 
  - 채용 담당자들의 검색 엔진 노출을 위해 시맨틱 HTML 작성 및 Meta Keyword 적절히 배치.
  - Lighthouse 기준 모든 지표 90점 이상 달성이 목표.
